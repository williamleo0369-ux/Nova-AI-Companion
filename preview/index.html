Nova AI Companion

有记忆 · 有情感 · 有温度

Python 3.10+ AsyncIO SQLite OpenAI Compatible
🧠

双层记忆系统

短期记忆维护对话上下文，长期记忆使用 SQLite 持久化存储。LLM 自动从对话中提取关键信息。

💕

情感引擎

基于关键词的情绪检测，识别 8 种情绪状态。动态调整 Nova 的心情、语气和回应策略。

🔊

语音系统

Edge-TTS 免费语音合成，支持朗读回复。预留 GPT-SoVITS 自定义音色的扩展接口。

🔧

工具系统

内置时间查询、计算器、心情记录等工具。支持 OpenAI Function Calling 标准格式。
